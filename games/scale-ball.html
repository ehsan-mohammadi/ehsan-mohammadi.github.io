Scale Ball - Game by Ehsan Mohammadi

SCALE BALL

Save the ball from spikes!
Change the ball size to pass through the way.
Scale Ball is an endless hyper-causal Android game which I made it with Unity in 2021. I developed it for Sharif Game Challenge (hosted by Sharif University of Technology) competition and I was awarded as 3rd best game between 129 teams (+1200 people).
HOLD to change the ball size BIGGER and RELEASE to change it SMALL. Be aware that don't touch the spikes!
