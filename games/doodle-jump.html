Doodle Jump - Game by Ehsan Mohammadi

DOODLE JUMP

How high can you get?!
Journey up a sheet of graph paper and perpetually jump from one platform to the next. This game is insanely addictive!
This is a remake version of Doodle Jump. An endless platforming Android game which I made it with Unity in 2018. I developed it during a class to teach my students "How to make a game with Unity" at Birjand University of Technology. I published my remake version source code, as a public GitHub repository.
Use mobile ACCELERATION-SENSOR to move the player in LEFT/RIGHT directions.
